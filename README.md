# 🍽️ Дүүрэн Ресторан — Next.js Website

Дархан-Уул аймгийн Дүүрэн рестораны вэбсайт. Next.js 15, TypeScript дээр бүтээгдсэн.

## Суулгах

```bash
npm install
npm run dev
```

Браузерт [http://localhost:3000](http://localhost:3000) нээнэ үү.

## Технологи

- [Next.js 15](https://nextjs.org/) — React framework
- TypeScript
- CSS Variables (Custom styling)

## Deploy

### Vercel (Шууд)

```bash
npm install -g vercel
vercel
```

### GitHub Pages / Vercel автомат

GitHub-д push хийхэд Vercel автоматаар deploy хийнэ.
